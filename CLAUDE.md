# Bauhaus Clock

## What This Is

A macOS screensaver recreation of [bauhausclock.com](https://bauhausclock.com/). Built as a native Swift `.saver` bundle using Core Graphics for all rendering.

## Architecture

```
ClockScreenSaverView.swift   # ScreenSaver.framework entry point, animation loop
  └─ ClockRenderer.swift     # Pure CGContext rendering engine
  └─ Palettes.swift          # NSColor palette definitions
  └─ ScreenSaverDefaults     # Persistent user preferences
```

Settings are stored via `ScreenSaverDefaults` (macOS system preferences):

```swift
"dial":     "Noir"          // active palette
"movement": "Mechanical"    // Quartz | Mechanical | Digital
"size":     "Classic"       // Classic (480px) | Compact (360px)
"night":    false
"lume":     "Tritium Green"
"seconds":  true
```

## Build Commands

### Build .saver (no Xcode required)
```bash
./build.sh
# → build/BauhausClock.saver
```

### Install
```bash
open build/BauhausClock.saver
# macOS prompts to install → "Install for this user only"
# Then: System Settings → Screen Saver → Bauhaus Clock
```

Or copy manually:
```bash
cp -R build/BauhausClock.saver ~/Library/Screen\ Savers/
```

### Xcode project (optional)
```bash
xcodegen generate    # requires: brew install xcodegen
open BauhausClock.xcodeproj
```

## Features

- 16 dial color palettes (White, Turquoise, Glacier, Ocean, Tennis, Signal Blue, Salmon, Yellow, Beige, Pistachio, Lavender, Rose, Sky Blue, Cream, Slate, Noir)
- 3 movement types: Quartz (1Hz tick), Mechanical (4Hz/28,800 bph), Digital (smooth continuous)
- Night mode with 9 lume colors (Tritium Green, Amber, Swiss BGW9, Ice Blue, Red, Lavender, Rose, Seafoam, White)
- Burn-in protection: random ±4px position drift every 45s via `Timer.scheduledTimer` + CGContext translate
- Settings panel: slide-in from right on click
- Classic (480px) / Compact (360px) size toggle
- Second hand toggle

## Design Decisions

### Hands
- **Shape**: Leaf/lozenge bezier path — pointed tip, widest at 38% down, tapered base
- **Rendering**: Outer border layer (scaled 1.12x) + inner metallic fill using 6-stop horizontal linear gradient (`metalEdge → metalMid → metalHi → metalHi → metalMid → metalEdge`)
- **Center highlight**: Faint line for chrome reflection
- **Shadows**: CGContext shadow, dx:1.5 dy:3 blur:3 opacity:0.35

### Hour Indices
- **Shape**: Capsule/stadium rounded rects at all 12 positions (including 12 and 6 o'clock)
- **Rendering**: Same 2-layer border + metallic gradient as hands
- **Shadows**: dx:0.8 dy:1.5 blur:1.2 opacity:0.3
- **Position**: `fR * 0.91` from center, length `fR * 0.12`, width 4.5px

### Minute Ticks
- 48 lines (skip positions 0, 5, 10, …55 — those are hour indices)
- Gray, 1.3px wide, 50% opacity
- Between `fR * 0.94` and `fR * 0.98`

### Face / Background
- Face color = background color — no visible boundary, no bezel ring, no outer border
- Grain texture: deterministic LCG noise (`x * 1103515245 + 12345`), 3px steps, soft-light blend at 8% opacity

### Center Hub
- 5-layer concentric rings: outer (8px, `capOuter`), mid (6px, `capMid`), groove (4.5px, `capInner`), bright dot (2.5px, `metalHi`), tiny dark center (1px, `capOuter`)

### Typography
- **Font**: Jost Medium (bundled in `BauhausClock/Resources/`) via CoreText — geometric Bauhaus-era typeface. Fallback: Helvetica Neue.
- **Size**: `sz * 0.052`
- **Position**: `fR * 0.74` from center

### Time Calculation (all in degrees)
- Hour hand: `h * 30 + m * 0.5 + s * (0.5/60)`
- Minute hand: `m * 6 + s * 0.1`
- Second hand:
  - Quartz: `s * 6` (discrete 1Hz ticks)
  - Mechanical: `floor((s*1000+ms)/125) * 0.75` (4Hz quantization)
  - Digital: `s * 6 + (ms/1000) * 6` (fully smooth)

### Night Mode
- All black background (`#080808`)
- Hands/indices rendered in lume color with gaussian blur glow (no drop shadows)
- Night palette computed from lume color base

## Palette Structure

Each of the 16 dials defines these color keys in `Palettes.swift`:

```swift
bg          // background + face (same color — no bezel)
idx         // minute tick color
num         // numeral color
sec         // second hand color
metalEdge   // dark edge of metallic gradient
metalMid    // mid tone of metallic gradient
metalHi     // highlight center of metallic gradient
border      // hand/index outer stroke
capOuter    // center hub outer ring
capMid      // center hub mid ring
capInner    // center hub groove ring
```

## Rendering Pipeline (ClockRenderer.swift)

Per frame via `draw(_ rect: NSRect)`:
1. Fill background with palette `bg`
2. Draw grain texture (deterministic LCG noise)
3. Calculate hand angles from current time
4. Draw 48 minute ticks
5. Draw 12 hour indices (2-layer capsule: border → metallic fill)
6. Draw numerals (CoreText, Jost-Medium, radius `fR * 0.74`)
7. Draw hour + minute hands (border layer → metallic gradient → highlight line)
8. Draw second hand if enabled (tail + counterweight circle + needle)
9. Draw center hub (5 concentric circles)

Scale factor: `scale = min(width, height) / clockSize` — all dimensions scale by this.

## What Still Needs Work

- **Typography**: Real v2 uses a custom in-house typeface. Jost is an approximation.
- **Hand geometry**: Bezier curves are close but real hands have more refined tapering (possibly concave edges).
- **Color accuracy**: All 16 palettes approximated from marketing thumbnails, not extracted from the binary.
- **Settings UI**: Real app has a polished native macOS settings sheet. Current is basic.
- **Languages**: Real app supports 10 languages (EN, FR, DE, ES, IT, JA, ZH, RU, TR, AR). Not implemented.
- **60 numeral**: Original shows "60" at 12 o'clock minute position in some modes.
- **Day/Night auto-switching**: Should follow `NSAppearance` / system theme.
- **Code signing**: Needs Developer ID + notarization for distribution.
